# GTA V/Online PS5 Enhancement Tool

Internetowe narzędzie do ulepszania doświadczenia gry w GTA V i GTA Online na PS5 z integracją sztucznej inteligencji (Gemini API).

## Funkcjonalności

- 🎮 **Integracja PS5** - pobieranie danych z konsoli PlayStation 5
- 🤖 **Chat z Gemini** - asystent AI do porad, strategii i wskazówek do GTA
- 📊 **Dashboard** - monitorowanie statystyk gracza i osiągnięć
- 💡 **Poradniki** - dynamiczne porady oparte na AI dla konkretnych misji
- 🎯 **Optymalizacja** - sugestie do usprawnienia rozgrywki

## Technologia

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express
- **AI**: Google Gemini API
- **PS5**: PlayStation Network API
- **Database**: MongoDB (opcjonalnie)
