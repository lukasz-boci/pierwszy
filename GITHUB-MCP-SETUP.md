# GitHub MCP Server Setup Guide\n\n## 🚀 Quick Start na Linux\n\n### 1. Uruchom skrypt instalacyjny\n\n```bash\nchmod +x install-github-mcp.sh\n./install-github-mcp.sh\n```\n\nLub ręcznie:\n\n### 2. Zainstaluj dependencje\n\n```bash\n# Upewnij się że masz Node.js 18+\nnode --version\nnpm --version\n\n# Jeśli nie masz Node.js\ncurl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -\nsudo apt-get install -y nodejs\n```\n\n### 3. Sklonuj GitHub MCP Server\n\n```bash\nmkdir -p ~/.local/share/claude/mcp-servers\ncd ~/.local/share/claude/mcp-servers\n\ngit clone https://github.com/modelcontextprotocol/servers github-mcp\ncd github-mcp\n\nnpm install\nnpm run build\n```\n\n### 4. Pobierz GitHub Token\n\n1. Wejdź na: https://github.com/settings/tokens/new\n2. Kliknij \"Generate new token (classic)\"\n3. Wybierz uprawnienia:\n   - `repo` (full control of repositories)\n   - `read:user` (read user profile)\n   - `read:org` (read organization data)\n4. Skopiuj token\n\n### 5. Ustaw zmienną środowiskową\n\n```bash\n# Dodaj do ~/.bashrc lub ~/.zshrc\nexport GITHUB_TOKEN=\"your_token_here\"\n\n# Załaduj zmienne\nsource ~/.bashrc  # lub source ~/.zshrc\n```\n\n### 6. Skonfiguruj Claude\n\nTwórz plik: `~/.config/claude/config.json`\n\n```json\n{\n  \"mcpServers\": {\n    \"github\": {\n      \"command\": \"node\",\n      \"args\": [\n        \"~/.local/share/claude/mcp-servers/github-mcp/dist/index.js\"\n      ],\n      \"env\": {\n        \"GITHUB_TOKEN\": \"${GITHUB_TOKEN}\"\n      }\n    }\n  }\n}\n```\n\n### 7. Zrestartuj Claude Desktop\n\nZamknij i otwórz Claude na nowo.\n\n---\n\n## ✅ Sprawdzenie czy działa\n\nW Claude spróbuj:\n```\nWyświetl moje repozytoria GitHub\n```\n\nLub:\n```\nPrzejrzyj issues w repozytorium lukasz-boci/pierwszy\n```\n\n---\n\n## 📋 Dostępne komendy GitHub MCP\n\n- **Przeglądaj repozytoria** - Lista Twoich repo\n- **Odczytaj pliki** - Zawartość plików z repo\n- **Zarządzaj issues** - Tworzenie, edytowanie issues\n- **Przeglądaj pull requests** - Historia PR\n- **Aktualizuj kod** - Push zmian do repo\n- **Zarządzaj gałęziami** - Tworzenie/usuwanie gałęzi\n\n---\n\n## 🔧 Troubleshooting\n\n### Problem: \"GITHUB_TOKEN not found\"\n\n```bash\n# Sprawdź zmienną\necho $GITHUB_TOKEN\n\n# Jeśli pusta, ustaw ją\nexport GITHUB_TOKEN=\"your_token\"\n\n# Lub dodaj do .bashrc\necho 'export GITHUB_TOKEN=\"your_token\"' >> ~/.bashrc\nsource ~/.bashrc\n```\n\n### Problem: \"Node.js not found\"\n\n```bash\ncurl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -\nsudo apt-get install -y nodejs\n```\n\n### Problem: \"Permission denied\"\n\n```bash\nchmod +x install-github-mcp.sh\n```\n\n---\n\n## 🎯 Integracja z projektem GTA\n\nTeraz możesz:\n- Zarządzać repo z Claude\n- Commitować zmiany\n- Tworzyć branches\n- Zarządzać issues\n- Przeglądać pull requests\n\n**Przykład:**\n```\nDodaj nowy feature branch o nazwie 'gemini-enhancement' w repo lukasz-boci/pierwszy\n```\n\n---\n\n## 📚 Dokumentacja\n\n- GitHub MCP: https://github.com/modelcontextprotocol/servers\n- Claude: https://claude.ai\n- GitHub Tokens: https://github.com/settings/tokens\n"
